# ArticleCrawler
文章自动爬取程序

### 相关版本
> 版本：开发版本0.2 [未发布]  
Python版本：3.7.5

### 依赖
> *requests* 网络请求库  
*bs4* HTML分析库

#### 可通过pip下载
``` batch
pip install --user -i https://pypi.tuna.tsinghua.edu.cn/simple/ requests  
pip install --user -i https://pypi.tuna.tsinghua.edu.cn/simple/ bs4
```
